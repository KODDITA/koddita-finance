# Koddita Finance 💙

Aplicación de finanzas personales con diseño futurista inspirado en Revolut. Construida con React, Vite y Recharts.

🔗 **Demo en vivo:** [koddita-finance.vercel.app](https://koddita-finance.vercel.app)

---

## ✨ Funcionalidades

- 📊 Dashboard anual con gráficas de barras y tendencia del balance
- 🍩 Gráfico de dona por categoría de gasto
- 💱 Multimoneda (EUR, USD, GBP)
- 🎯 Objetivos de ahorro con barra de progreso
- 📋 Historial completo con búsqueda y filtros
- ⚠️ Límite mensual con alertas visuales
- 🌙 Modo claro / oscuro
- 📄 Exportar a PDF
- 💾 Datos persistentes con localStorage
- 🎬 Splash screen animado con logo

---

## 🛠️ Tecnologías

- React 19
- Vite
- React Router DOM
- Recharts
- Lucide React
- CSS personalizado con variables y efectos neón

---

## 🚀 Instalación local

```bash
git clone https://github.com/KODDITA/koddita-finance.git
cd koddita-finance
npm install
npm run dev
```
